# Quiz SaaS App

A modern quiz platform built with **Next.js** for creating, sharing, and analyzing quizzes.

## Live Demo

[https://quiz-saas-app.onrender.com](https://quiz-saas-app.onrender.com)

## Features

- Create and edit quizzes with multiple questions and options  
- Submit quizzes and get instant scoring  
- View quiz analytics and detailed statistics  
- User authentication and personal dashboard  

## Getting Started

Clone the repository and install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Run the development server:

```
npm run dev
# or
yarn dev
# or
pnpm run dev
```

open in your browser:

http://localhost:3000

The app is deployed on Render:

https://quiz-saas-app.onrender.com

Tech Stack:
- Next.js
- Supabase (Authentication & Database)
- Tailwind CSS
